Edge-of-The-Empire-Dice
=======================

A dice rolling tool for the Edge of the Empire roll playing game.

####usage:
	python eote.py -p abcdpsf0


####results in:

	two success
	advantage
	failure
	40
	two advantage
	failure
	two light
	3

the string after `-p` signifies which, and how many dice are rolled.

	a = ability
	b = boost
	c = challenge
	d = decader(d10)
	p = proficiency
	s = setback
	f = force
	0 = d10
