Edge-of-The-Empire-Dice
=======================

A dice rolling tool for the Edge of the Empire roll playing game.

####usage:
	python roll.py -p abcdpsf10


####results in:

	blank
	blank
	blank
	threat
	success + advantage
	threat
	two light
	10
	5

the string after `-p` signifies which, and how many dice are rolled.

	a = ability
	b = boost
	c = challenge
	d = Difficulty
	p = proficiency
	s = setback
	f = force
	1 = 10-90
	0 = 1-9
